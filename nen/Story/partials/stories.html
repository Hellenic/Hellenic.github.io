Storytime

Interactive storybook

This is a interactive storybook. With the Storybook it's possible to make stories where user doesn't just read the stories but also takes part in them.

In story it's possible to ask user some questions and use the answers later as a part of the story. Also you can make some puzzles what user has to solve to get forward. User can also make choices how to go forward, so the story can have different branches.
