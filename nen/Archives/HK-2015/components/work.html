My works

This page contains links to all kinds of stuff I've done. Most of the projects are done just to learn more about that language or technique.

Games

Cardeon
Browsed-based card game made with JavaScript and HTML5, using technologies such as WebGL and Websocket. Server-side written with Java using Spring MVC & DI, Hibernate and Bristleback for Websocket. Database in MySQL and webapp running on Jetty.
NatureQuiz
Quiz game for Android to test your knowledge on most common nature-related things! Game will show you pictures and you need to write in the answer. Read more about the game in a blog post. Source code you can find on Github.

Web version on the works as well; NatureQuiz web app.
