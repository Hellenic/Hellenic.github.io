Information

This menu system is still under development. It might not render correctly on every browser and it definitely does not look very beautiful... Yet.

Also what I intend to do here might not be the best option from usability point of view. Mostly I wanted to do something new and different, what is not seen on web that much yet. Modern browsers are capable of so much these days and this is not brought up enough.

Overall I think it looks kinda nice and cool. Rather easy to use (except the text is not very easy to read yet). Also big buttons so it works nicely on mobile too.

This page is powered by Bootstrap (which is not visible if the menu is renderable in your browser) and Three.js which, hopefully you are seeing in use right now.
